# 업무천재 고주무관 (업천고)

학교 행정실의 반복업무를 백엔드 없이 브라우저에서 끝내는 웹도구를 만듭니다.

강원특별자치도 공립학교 행정실에서 근무하는 교육행정직 말단 공무원입니다.
- 담당 업무: 보수·인건비, 복무, 행정서무 등
- 관심사: 웹개발, 정보보안

**도구 모음: https://edusprouthaey.co.kr**

## 설계 원칙

- 단일 HTML + 백엔드 안 씀. 외부 의존성 없음 — 학교 현장 어디서든 열립니다
- 100% 브라우저 내부 처리 — 파일과 데이터가 서버로 전송되지 않습니다 (서버가 없거든요)
- 비공식 보조도구임을 명시 — 최종 기준은 항상 공문·지침·편람 등 공식자료를 우선으로 사용자 본인이 판단하셔야 합니다.
